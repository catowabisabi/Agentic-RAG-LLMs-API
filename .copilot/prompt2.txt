bash (window 0) - 可以保留作為主控制台，用於執行腳本和管理
重複 windows - 是之前腳本執行多次造成的，清理腳本會移除

Prompt 指令 - 已更新 memory.json，以後：
使用 window 編號 (0,1,2,3) 而不是名稱
Window 2 = api_terminal (Port 1130)
Window 1 = testing_terminal (測試用)


標準化指令模式：
# 啟動服務
tmux send-keys -t agentic_rag:2 'python3 main.py --ui' Enter

# 檢查狀態  
tmux capture-pane -t agentic_rag:2 -p | tail -10

# 列出 windows
tmux list-windows -t agentic_rag


這樣就不會再有名稱混亂問題！
