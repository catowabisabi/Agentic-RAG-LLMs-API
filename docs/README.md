# 文檔中心

此文件夾包含所有項目相關文檔，已按類別組織。

## 📁 文件夾結構

### 📂 project/
項目整體文檔和規劃
- `PLANNING.md` - 項目規劃和路線圖
- `project-folder-structure.md` - 項目文件夾結構說明
- `PROJECT_README.md` - 項目總覽和介紹
- `SYSTEM_DOCUMENTATION.md` - 系統架構文檔
- `UPGRADE_SUMMARY.md` - 系統升級總結

### 📂 guides/
使用指南和操作手冊
- `SMART_RAG_USAGE.md` - Smart RAG 使用指南
- `SMART_RAG_STATUS.md` - Smart RAG 狀態說明

- `VECTOR_DATABASE_GUIDE.md` - 向量數據庫操作指南

### 📂 api/
API 相關文檔
- `MCP_INVESTOR_GUIDE.md` - MCP (Model Context Protocol) 投資者指南

### 📂 refactoring/
代碼重構相關文檔
- `REFACTORING_PLAN.md` - 完整重構計劃（5 個階段）
- `PHASE_1_COMPLETION.md` - 階段 1 完成總結

## 🔍 快速導航

### 新開發者
1. 閱讀 [PROJECT_README.md](project/PROJECT_README.md) 了解項目
2. 查看 [project-folder-structure.md](project/project-folder-structure.md) 理解結構
3. 參考 [SYSTEM_DOCUMENTATION.md](project/SYSTEM_DOCUMENTATION.md) 學習架構

### 使用者
1. 查看 [SMART_RAG_USAGE.md](guides/SMART_RAG_USAGE.md) 學習如何使用
2. 參考 [TOKEN_USAGE_GUIDE.md](guides/TOKEN_USAGE_GUIDE.md) 控制成本
3. 閱讀 [VECTOR_DATABASE_GUIDE.md](guides/VECTOR_DATABASE_GUIDE.md) 管理數據

### 開發新功能
1. 閱讀 [HOW_TO_CREATE_NEW_AGENT.py](refactoring/HOW_TO_CREATE_NEW_AGENT.py)
2. 參考 [REFACTORING_GUIDE.py](refactoring/REFACTORING_GUIDE.py) 了解最佳實踐
3. 查看 [REFACTORING_PLAN.md](refactoring/REFACTORING_PLAN.md) 了解架構方向

### API 集成
1. 查看 [MCP_INVESTOR_GUIDE.md](api/MCP_INVESTOR_GUIDE.md)
